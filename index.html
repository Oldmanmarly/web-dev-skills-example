WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Languge. HTML is the content of a web page. Examples of HTML include headings, paragraphs, images, links, lists etc.

Check out the HTML tutorial on w3schools.com.

Learn CSS

CSS stands for Cascading Style Sheet. CSS is the style of a web page. Examples of CSS include font styles and color, background color, borders, dimensions, layout, etc.

check out the CSS Tutorial on w3schools.com.

Learn JavaScript

JavaScript is the programming languge of the web. JavaScript helps to make web pages interactive and dynamic. Some common JavaScript tasks are:

  • add new HTML to the page,
  • change the existing content / style,
  • react to user events (mouse clicks, key presses, etc.)

check out the JavaScript tutorial on javascript.info.
